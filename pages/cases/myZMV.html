MyZMV

CRM
Web app

Project
Overview

The system is a software for a dental company in Germany. It's a CRM that provides a full range of tools and functions for managing patients starting from their first visit and ending with postoperative care.

go to the website
  • Healthcare Tech
  • CRM
  • 5 months

Project Goals

  • Clear Visual Component /
  • Cost Calculation /
  • Patient Reporting /

The system should be understandable both for medical professionals and ordinary patients. According to predefined formulas and a previous price list, the system should generate several cost calculation options. Generating a clearly described PDF report about the patient's dental problem, treatment course, and price.

Solutions

  • Presenting Services to Clients /
  • Online Cost Calculations /
  • Technical Reporting /

Ability for Doctors to Present Their Services to Clients. Through effective UX/UI design, we managed to create a user-friendly interface that was understandable for both doctors and patients. Using state-of-the-art technologies, we were able to create an online calculator that helped calculate various service cost options and also generate additional individual subsidies. Under the hood, the system has complex scripts that analyze existing prices and compare cases, based on which a clearly formed PDF report for the client is generated.

Technologies Used

Challenges

While talking to the patient, the dentist should be able to show real-time calculations of various options, and the patient can see and choose the best one for them in terms of cost and quality. To achieve this, we utilized intuitive UX/UI and online calculation.

Additionally, the tool should have a table similar to Excel for managing treatment options, where data can be modified, for dentists and related staff.

All rows and content of the table should have cells linked to an API that receives the latest changes, ensuring that the user always has the most up-to-date version of treatment types and prices. This was made possible through the use of cutting-edge technologies.

Project phases

The Roadmap

This project was built by our team at every stage. We conducted thorough exploration during the discovery phase, where we discovered many interesting ideas for the design phase. After successful design approval, we were able to smoothly transition into the development phase, where our technical experts delivered excellent results. As a result, the testing and bug-fixing phase proceeded smoothly and quickly.

Phase 1

Research & Design

In project development, it is always important to pay attention to the research stage, as it allows for a deep understanding of the client's idea and identifies important milestones in project formation. During the design phase, we contemplated the overall medical concept of the project formation, as the specifics of the dental environment are unique. Therefore, we were able to meet the needs of both medical staff and dental clinic clients successfully.

Phase 2

Development & Testing

It is worth noting that medical data is highly sensitive, so we paid great attention to ensuring a secure and accurate format for storing and transmitting data. Another valuable aspect was that we managed to select the appropriate API that did not overload the system but instead provided flexibility to transmit information quickly and smoothly. A significant aspect of forming PDF reports, which essentially serve as the core for further sales in the hospital, was successfully implemented into the project.

Phase 3

Maintenance

We skillfully handled customer support for two and a half years. By implementing the project live, we were able to receive feedback from real users, which allowed us to improve the overall logic and appearance of the website and add additional functionality.

Conclusion

Summary

The My ZMV project, which we developed using Laravel and Vue.js technologies, proved to be interesting and successful among end users. This allowed our client to expand their business and improve communication and sales among patients. We are pleased with the close collaboration with the client, who provided clear tasks and consulted with us on project implementation.
